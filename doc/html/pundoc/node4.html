Limitations

  1. In The World's Writing Systems, Bright and Daniels, a character a is given. This character is la with a nuqta beneath it. There is no glyph for it in the pun font. Perhaps it can be hard-coded.

  2. The symbol of the Sikhs, ka, is available as a glyph in the pun font. This is accessible as khNDa or KND in the input text.
